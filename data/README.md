# Data

This directory is intentionally empty in the repository.

## Why no data is committed

The IPUMS USA Usage License prohibits redistribution of raw microdata. This
dashboard's source data (the IPUMS extract) is therefore not committed here.
See [ipums/extract-spec.md](../ipums/extract-spec.md) for how to reproduce
the extract.

## What goes where

| Path | Purpose | Committed? |
|---|---|---|
| `D:\IPUMS\` (or wherever you save it) | Raw IPUMS extract | No — outside repo, gitignored regardless |
| `data/ipums/` | (Convention only — never used) | No — gitignored |
| `data/aggregated/` | Output of `scripts/ipums_aggregator.py` | Optional — small aggregated CSVs only |

The Power BI report reads `data/aggregated/medians_by_country_age_year.csv`,
generated by the aggregator. That file is small (~50 KB) and contains only
medians + counts — no individual-level data. Committing it is permitted.
